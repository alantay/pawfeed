# 🐾 PawFeed

> The frictionless window into your pet's stay.

PawFeed is a mobile-first web application designed for pet sitters to provide real-time updates to owners with zero friction. No owner accounts, no app store downloads—just private "Magic Links" to a beautiful, curated pet timeline.

## ✨ The Vision

Traditional pet boarding updates happen over fragmented chat threads. PawFeed centralizes the experience into a professional "Story" feed, allowing sitters to focus on the animals while keeping owners connected through a "Photo-First" workflow.

## 🚀 Key Features

- **Frictionless Posting:** "Input-First" design—take a photo and post in two taps.
- **Magic Links:** Unique, unguessable URLs for owners to view their pet's feed securely without logging in.
- **Smart Context:** Automatic, time-of-day based titles (Morning, Afternoon, Evening) to reduce typing.
- **Privacy-First:** No sensitive contact details or personal data displayed on public-facing links.
- **Mobile Optimized:** Built as a PWA-ready web app to feel like a native "point-and-shoot" experience.
